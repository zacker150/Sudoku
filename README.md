# Sudoku
Demonstrates solving a Sudoku problem using an optimized backtracking algorithm. 

Soduku boards are stored in files as 9x9 grids of integers

Example of a soduku grid imported from file. 

	0 0 0 0 4 0 0 2 6 
	0 4 0 5 6 0 7 0 0 
	0 0 0 0 0 3 1 5 0 
	0 2 0 0 0 8 5 0 0 
	7 0 0 4 0 5 0 0 2 
	0 0 3 1 0 0 0 9 0 
	0 7 8 9 0 0 0 0 0 
	0 0 4 0 2 7 0 8 0 
	2 5 0 0 1 0 0 0 0
